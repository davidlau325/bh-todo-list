# bh-todo-list

A multi-window To-do list app built with Electron, React and Redux

## Getting Started
### Prerequisties
```
node > v5.0 
npm > v3.3.0
```
### Installing
Run npm install to grab all dependencies
```
$ npm install
```
### Run
run build and webpack
```
$ npm run build
```
and, in another terminal, 
run electron
```
$ npm start
```
